# Docker

- khái niệm: chương trình tạo môi trường ảo(container)

**- Dowload file-upload-example.zip rồi giải nén**

- Mở folder **file-upload-example trong visual studio**

- Bật docker desktop

- Vào visual studio mở terminal nhập: docker-compose up

![image.png](/images/image%2036.png)

→ chạy thành công

- docker-compose up//tạo container với config viết trong file docker-compose.yml

- Khi người dùng truy cập port 12000 ngoài máy thật thì tương ứng với port 80 trong container

![image.png](image%2037.png)

- Truy cập localhost:12000

![image.png](image%2038.png)

- var_dump($_FILES); //là 1 hàm dùng để debug, in ra kiểu dữ liệu và giá trị dữ liệu của biến
- docker ps //cho biết những container nào đang chạy kèm theo id
- docker exec -ti 7bd350c1ba12 bash //vào trong terminal trong container
